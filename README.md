# Diwali Sales Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Diwali Sales dataset to understand customer purchasing behavior.

## Objectives

- Clean the dataset
- Analyze customer demographics
- Understand purchasing patterns
- Visualize important insights

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Diwali Sales Dataset

## Key Insights

- Female customers purchased more than males.
- Customers aged 26–35 contributed the highest sales.
- Uttar Pradesh and Maharashtra generated the highest revenue.
- Married women from IT, Healthcare, and Aviation sectors were major buyers.
